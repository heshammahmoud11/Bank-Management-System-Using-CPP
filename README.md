# Bank-Management-System-Using-CPP
A C++ console application that performs several process to manage users and clients on the system like : Add new client, show clients list, delete client, find client, update client.
Also has a transaction feature help users to make money operatoins on their accounts like, withdraw, deposite,and show balance.
## New Features
I also added 2 new features "manage users" and "login screen"
each has different usage.
-Manage user feature : help users to make some process like list users, add user, delete user and etc.
-login screen : any one want to access to this system have to login in first with the right credintial to access it.
